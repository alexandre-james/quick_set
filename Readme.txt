"Quick sets" is a custom Maya panel that lets you easily manage multiple selections.
It lets you create an infinite amount of selection sets and recall them later. 
Selections can be created, modified, renamed or deleted in a single mouse click.
Can be used with all types of Maya objects.

Works with Maya 2017/2018

Installation:

1)Unzip "Sets_control" folder to your Maya scripts directory

"C:\Users\user_name\Documents\maya\Your_Maya_Version\scripts"

2)Launch the script with this Python command:

from Sets_control import set_control
set_control.show()
